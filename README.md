# 🔬 Global Antibiotic Consumption Analysis (Python)

## 📊 Overview
This project provides a comprehensive statistical and exploratory analysis of global antibiotic consumption data (DDD per 1000 inhabitants) based on WHO datasets(https://www.who.int/data/gho/data/indicators/indicator-details/GHO/antibacterial-consumption--total-consumption-of-antibacterials-expressed-as-ddd-per-1000-inhabitants-per-day). The analysis is conducted using Python and focuses on identifying trends, correlations, regional differences, and variability across countries.

---

## 🎯 Objectives
- Analyze global antibiotic consumption trends over time
- Evaluate relationships between time (Year) and consumption (DDD)
- Compare consumption patterns across regions
- Identify statistically significant differences between regions
- Assess variability and stability at the country level

---

## 📁 Dataset
The dataset includes
- Country-level antibiotic consumption (DDD per 1000 inhabitants)
- Yearly observations
- Regional classifications (WHO regions)

---

## 🛠 Tools & Libraries
- Python (Pandas, NumPy)
- Visualization Matplotlib, Seaborn
- Statistical Analysis SciPy (Pearson, Spearman, Mann-Whitney U)
- Google Colab
---

## 🔍 Analysis Workflow

### 1️⃣ Data Preparation
- Data loading and inspection
- Data type validation
- Missing value analysis

---

### 2️⃣ Correlation Analysis (Year vs DDD)
- Pearson and Spearman correlation tests
- Evaluation of global consumption trends over time

📌 Insight
- Identifies whether antibiotic consumption is increasing or decreasing globally

---

### 3️⃣ Advanced Regional Comparison
- Aggregation of consumption metrics by region
- Comparative visualization across regions

📌 Insight
- Highlights regions with highest and lowest consumption levels

---

### 4️⃣ Statistical Testing (Mann-Whitney U Test)
- Pairwise comparison between regions
- Non-parametric testing for distribution differences

📌 Insight
- Detects statistically significant differences in consumption patterns between regions

---

### 5️⃣ Country-Level Analysis
- Correlation analysis at country level
- Trend detection for individual countries

📌 Insight
- Identifies countries with increasing or decreasing consumption trends

---

## 🧠 Key Insights

- Global antibiotic consumption shows a positive trend over time
- Significant regional disparities exist across WHO regions
- Some regions exhibit statistically higher consumption levels
- Several countries show strong increasing or decreasing trends
- Variability across countries suggests heterogeneous consumption behavior
